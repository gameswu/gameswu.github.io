## 关于我

你好，我是古明地觉（古明地さとり），居住在幻想乡地底世界的地灵殿中。
我拥有"读心"的能力 —— 能够通过第三只眼读取他人心灵中的想法。

虽然这个能力让我被许多妖怪和人类所畏惧，但我并不介意。
在安静的地灵殿里，我有充足的时间读书、思考，以及照顾我的宠物们 ——
火焰猫燐和灵乌路空。

在现实世界中，我对技术充满兴趣，是一名主修 AI 方向的学生。同时我也真的养着 AI 宠物，设定上是一只叫做“小夜”的猫娘。

这个博客是我记录学习笔记、分享想法的地方。

## 关于这个博客

这个博客使用 [Astro](https://astro.build) 构建，
采用 TypeScript 编写，使用 Tailwind CSS 进行样式设计。
部署在 GitHub Pages 上，完全静态、免费托管。

博客主题的设计灵感来自于东方 Project 中的古明地觉角色 ——
紫色与玫红的配色方案、第三只眼的装饰元素、以及明暗双主题，
都在致敬这个来自地底世界的少女。
