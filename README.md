# Satori Blog - 觉的博客

基于 [Astro](https://astro.build) 构建的个人博客，以东方 Project 古明地觉为主题。

## 快速开始

```bash
npm install
npm run dev
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。

## 写文章

在 `src/content/posts/` 目录下新建 `.md` 文件：

```markdown
---
title: '文章标题'
description: '文章描述'
date: '2025-01-01'
tags: ['标签1', '标签2']
---

正文内容...
```
